# TravelUP





- User Stories: 

As a developer, I want to extend the data model so that a Quote can include hotels with name, cost per night, 
and total cost, as well as flights with departure and arrival airports, flight number, dates, and cost, 
so that the system reflects detailed travel options. Adapt existing screens accordingly. (X)

- User Stories: 

As a developer, I want to update the data model to include city and country information, and create screens and logic to upload cities and countries from a CSV file, 
so that location data is comprehensive and manageable.

- User Stories:

As a developer, I want to adapt existing screens to use city and country data from the database, and implement dropdowns to select first the country and then the city, so that location selection is user-friendly and accurate.
